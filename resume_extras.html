  • Radio Automation

    Demo Code

      Radio automation and scheduling software as an open-source project for my college radio station, KUPS 90.1FM The Sound. The software uses information from music files to datamine exact music metadata from online sources, and also finds currently-trending music to add to the rotation. After validating music files with this metadata, the software queries these same sources for several metrics of popularity (enabling the software to rank the frequency with which the music should be played) as well as for genres, subgenres, and similar artists. Using these various classification features, playlists of similar music are generated using a k-nearest-neighbors approach. Take a look at some example playlists here.

  • Personal Organizer / Workload Balancer

    Code

      An organizer program that optimizes your work time on projects with designated due dates and estimated working hours required. Optionally, can weight different dates/days of the week to have more/less work than others. Typical usage without such weights results in work being as evenly distributed as possible within the time constraints of the schedule. Essentially, the program is solving the optimization problem "minimize the variance of the hours per day of work (divided by weight), while keeping to certain time constraints for finishing certain work by certain dates", using a Random-restart hill climbing algorithm.
